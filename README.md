# E-Commerce
MERN stack web development project that enables users to purchase video games. Upon successful payment user will be able to down invoice in PDF format.
•	Developed an E-Commerce website for selling video games using frontend technologies such as React.js, JavaScript (JS), CSS and Axios, and backend technologies such as Node.js, Express.js and Stripe.
•	Provided 5 essential username, password and payment amount validations on screen using libraries such as Formik and Yup.
•	Implemented a dummy payment gateway integration using Node.js and Stripe API.
•	Using jsPDF library, user will be able to download invoices in PDF format after successful payment.
•	Implemented 3 reusable JavaScript (JS) functions and React.js components to avoid repetitive creation and stylization of product catalogue.
